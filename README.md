# C++ Car Rental System

## Description
This project is a console-based Car Rental System developed in C++. It demonstrates Object-Oriented Programming (OOP) concepts such as classes, encapsulation, and the use of STL vectors.

The system allows users to manage cars in a rental service and perform operations such as adding cars, renting cars, returning cars, and searching for available vehicles.

## Features
- Add new cars to the system
- Display all available cars
- Rent a car
- Return a rented car
- Search for cars by ID
- Delete cars from the system

## Concepts Used
- Object-Oriented Programming (OOP)
- Classes and Objects
- Encapsulation
- STL Vector
- Menu-driven programming
